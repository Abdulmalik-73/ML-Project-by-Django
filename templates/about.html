About This Project

The Haramaya House Price Prediction System is a machine learning-powered web application designed to provide accurate house price estimates for properties in Haramaya Town, Eastern Ethiopia.

This system leverages advanced regression algorithms to analyze various house features and predict market prices with high accuracy.

How It Works
1. Data Collection

We collected data on house sales in Haramaya Town including features like:

  • Number of bedrooms and bathrooms
  • House and land size measurements
  • Location within Haramaya Town
  • House condition (New, Good, Needs Renovation)
  • Year the house was built
2. Model Training

We trained a Random Forest Regression model on this data. Random Forest is an ensemble learning method that combines multiple decision trees to make robust predictions.

3. Prediction

When you submit house details, the model analyzes the features and generates a price prediction along with a confidence range.

Machine Learning Basics
What is Machine Learning?

Machine Learning is a subset of artificial intelligence that enables systems to learn and improve from experience without being explicitly programmed.

Regression Models

Regression is a supervised learning technique used to predict continuous values (like prices). Our model learns patterns from historical data to make predictions on new data.

Random Forest

Random Forest is an ensemble method that creates multiple decision trees and combines their predictions. This approach reduces overfitting and improves accuracy.

Model Evaluation

We evaluate our model using metrics like:

  • RÂ² Score: Measures how well the model explains variance (0-1, higher is better)
  • MAE: Mean Absolute Error - average prediction error in ETB
  • RMSE: Root Mean Squared Error - penalizes larger errors more heavily
Data Sources

The training dataset consists of 56 house sales records from Haramaya Town, covering:

  • 6 different locations in Haramaya Town
  • Houses built between 2008 and 2022
  • Various house conditions and sizes
  • Price range: 380,000 - 1,600,000 ETB

Note: This is a demonstration system with sample data. For production use, more comprehensive data collection would be recommended.
